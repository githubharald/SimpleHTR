Home

The Text Transcriber is intended to help the visually impaired, students, and those who need an efficient way to transcribe handwritten words by offering a machine learning model that can translate handwritten images to text and speech.

  • A drop down menu to select what file you want to run through the model
  • Using a Word Seperator to first find all the words in the sentence(s)
  • Using a Text-Transcriber AI to print out the text and combine them
  • Using Google's Text-to-Speech API to read out the transcribed text

Model

Our model consists of a NN with 5 CNN layers and 2 RNN layers. The CNN layers are responsible for extracting relevant features from the image, while the RNN layers propagates the relevant information and maps it to a matrix. The matrix is then sent to a Connectionist Temporal Classification (CTC) layer. This decodes the matrix and outputs the final text.
